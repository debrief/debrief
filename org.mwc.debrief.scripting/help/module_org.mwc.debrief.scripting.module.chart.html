Chart Module

capabilities related to managing the chart

Method Overview

Method Description
add4WGrid() Function that adds a 4WGrid to the current plot.
addETOPO() Function that adds a ETOPO to the current plot.
addGrid() Function that adds a Grid to the current plot.
addLocalGrid() Function that adds a Local Grid to the current plot.
addScale() Function that adds a Scale to the current plot
addTimeDisplayAbsolute() Function that adds the time display (Absolute) to the current plot.
addTimeDisplayRelative() Function that adds the time display (Relative) to the current plot.
addVPTLayers() Function that adds the VPT Layers to the current plot.
loadChartLibrary() Function that loads the Chart Library defined in the maritime preferences to the current plot.
loadCoastLine() Function that adds a Coastline to the current plot.
loadNaturalEarth() Method that load the natural earth from the path specified in the maritime preference section.
